# Assignment-manager
College Assignment & Attendance Manager

A personal college management web app to track subjects, assignments, and attendance across all semesters.

Features

- Year → Semester → Subject navigation
- Assignment tracking for each subject
- Add, edit, and manage assignment details
- Semester timetable management
- Daily attendance marking (Present / Absent)
- Visual attendance status (green/red)

Tech Stack

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- SQLite
